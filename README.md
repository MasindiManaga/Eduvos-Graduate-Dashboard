# Eduvos Graduate Survey Analysis & Dashboard

## Project Overview
This project analyzes survey data from Eduvos graduates to explore trends in technology usage, employment, and industry placement. It demonstrates practical R programming skills in data cleaning, visualization, and interactive dashboard development using R Shiny and Quarto. Insights from this analysis can support curriculum improvements and align training with industry demands.

## Key Features
### Data Cleaning & Preparation: 
- Handling missing values, standardizing categorical data, and subsetting key variables.

### Data Analysis & Visualization:
- Frequently used tech tools (programming languages, databases, AI tools, web frameworks, cloud platforms)

- Popular industries and top job roles by study field

- Employment rates across different study fields

### Interactive Dashboard:
- Built with R Shiny + Quarto for dynamic exploration of tech tool usage and graduate trends.
